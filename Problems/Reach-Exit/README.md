# Reach Exit

Your given a 2D maze and you need to check if it is possible to reach the exit.

## Infos which are maybe needed

Each cell is one of the following symbols:

- `.` Empty space (Can walk here)
- `#` wall (Can not walk here)
- `@` start position
- `E` exit
